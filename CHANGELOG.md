# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of EPL Home Win Predictor
- Logistic Regression model with Elo ratings and form features
- Comprehensive backtesting framework
- Kelly Criterion money management (full, half, quarter variants)
- Risk analysis module with CVaR and Monte Carlo simulations
- Dynamic leverage optimization based on capital level
- Walk-forward validation for realistic backtesting
- Detailed documentation and API reference
- Unit tests for core functionality
- Example scripts for quick start and advanced usage

### Performance
- 66.6% prediction accuracy
- 79.5% win rate on filtered bets
- +12.8% ROI over 10-year backtest period
- 1.62 Sharpe ratio

### Technical Details
- Python 3.8+ compatible
- Dependencies: numpy, pandas, scikit-learn, matplotlib
- Installable via pip
